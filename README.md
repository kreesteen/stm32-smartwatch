# STM32 Smartwatch Prototype

## Overview
Developed a prototype smartwatch using an STM32 microcontroller for health monitoring applications, including early-stage fall detection concepts.

## Features
- Sensor input processing using ADC
- Interrupt-driven system behaviour
- Real-time event handling using timers

## My Contributions
- Interfaced peripherals including LCD, LDR, keypad, and motor
- Implemented interrupt-driven firmware using timers and external interrupts
- Debugged timing and responsiveness issues in embedded firmware

## Tech Stack
STM32, C, STM32CubeIDE

## Key Concepts
Embedded systems, real-time systems, hardware–software integration
